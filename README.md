# VMCA 官方网站

[English](README_en.md)

## 项目简介

这是澳洲万年市华人互助会（MACMAA）的官方网站项目。网站采用现代化的 React + TypeScript 技术栈构建，提供完整的社区服务功能，包括新闻动态发布、活动中心管理、会员申请系统等核心功能。

### 主要特性

- 完整的中英文双语支持
- 新闻动态发布与管理
- 活动中心与在线报名
- 会员申请与管理
- 现代化的响应式设计
- 基于 Vite 的快速开发体验

### 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **动画库**: Framer Motion
- **后端服务**: Supabase
- **部署平台**: Vercel

### 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
